llmango: Go-specific package implementation

How It Works

Define a goal in your mango internal package:

package mango

import "github.com/llmang/llmango"

type TesterInput struct {
    FirstString  string `json:"firstString"`
    SecondString string `json:"secondString"`
}

type TesterOutput struct {
    FirstSentence  string `json:"firstSentence"`
    SecondSentence string `json:"secondSentence"`
}

var testerGoal = llmango.Goal[TesterInput, TesterOutput]{
    GoalInfo: llmango.GoalInfo{
        UID:         "tester-two-sentences-two-strings",
        Title:       "Create Two Sentences From Two Strings",
        Description: "Create sentences using two strings in different orders.",
    },
    Validator: nil,
}

Run llmango create to generate type-safe functions, then use it in your code:

output, err := mango.DoGoal1(input)
